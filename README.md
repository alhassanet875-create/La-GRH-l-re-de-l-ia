# Conférence : La GRH à l’ère de l’IA

Ce projet est un site web permettant aux participants de s'inscrire à la conférence **La GRH à l’ère de l’IA**.

## Fonctionnalités

- Formulaire d'inscription en ligne
- Génération automatique d’un QR Code pour chaque participant
- Enregistrement des inscriptions dans Google Sheets
- Tableau de bord administrateur
- Statistiques des inscriptions

## Technologies utilisées

- HTML
- CSS
- JavaScript
- Google Sheets (base de données)
- GitHub Pages (hébergement gratuit)

## Pages du site

- `index.html` : Page d'inscription
- `admin.html` : Tableau de bord administrateur
- `style.css` : Style du site
- `script.js` : Gestion des inscriptions
- `admin.js` : Gestion du tableau de bord
